La experiencia es extraordinaria

Nuestro periodista Jean-Pierre Gagick prueba el último modelo de Porsche: el 911 S/T. Y para encontrar el origen del nombre "S/T", tenemos que remontarnos a la creación original en 1969. En aquel momento, el fabricante quería crear una versión más ligera del 911 S para competición. Ante este éxito inesperado, Porsche decidió crear su modelo homologado para circular.
